## Server Structure

**Folder Structure**

```
modules/
└── csharp-module.dll
resources/
└── my-example-csharp-resource/
    ├── Alt.Net.Example.dll
    ├── resource.cfg
    └── ... (any .dll dependency like "AltV.Net.dll" or "mysql.dll")
```

**resource.cfg**

```json
{
    "type": "csharp",
	"main": "AltV.Net.Example.dll"
}
```
